Current Usage

The usage of the above code uses a "/" as the pause between grid references and a "-" to differenciate between words.
There is also a space between words

The cipher

Tap Cipher, or Knock Code, is a way to encode text messages in a simple way.
The tap code was often used by prisoners by banging metal bars or pipes against the wall.
Each letter is sent via tapping two numbers. The first telling us the row and the second telling us the column.
for example to "tap" the letter B the prisoner would tap once, pause then tap twice.
When you use this code in the real world, the letter "x" would be used to represent a new sentence
and the letter "k" to represent acknolwedgements

Design

The code is designed based on a Polybius Square, wich is a 5x5 grid of letters.
Each character in the latin alphabet is represented except from "k" which shares it's spot with "c"
The message is sent via taps or knock sounds against the wall.

The need for the code

The tap code is most often used by prisoners over morse code due to the complexity of morse code.
For the prisoner the use of morse code causes s difficulty as it required two seperate sounds.
a "dot" and a "dash". Tap code is just a dot.
Further a morse code novice also would probably require a cheat sheet until they have learned the code
